description: Review and maintain grug-brain memories. Commits changes to git, finds cross-links, flags stale memories. Use with /loop for periodic maintenance.
allowed-tools: mcp__grug-brain__grug-dream, mcp__grug-brain__grug-write, mcp__grug-brain__grug-delete, mcp__grug-brain__grug-read, mcp__grug-brain__grug-search
---

Call grug-dream to get a dream report. Review the report and take action:

1. **Stale memories**: Read each stale memory with grug-read. Check if the content is still accurate. Update with grug-write if the info has changed, or delete with grug-delete if it's no longer relevant.
2. **Cross-links**: Review cross-linked memories. If they describe related concepts across categories, add a brief cross-reference note to each (e.g., "See also: category/memory-name").
3. **Quality issues**: Fix memories missing dates or descriptions by reading and rewriting them with grug-write.
4. **Inventory review**: Scan the full inventory for contradictions, duplicates, or memories that have been superseded by newer ones.

Be conservative. Only modify memories you are confident about. When unsure, skip.
